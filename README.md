# 📚 Projetos em Java

Este repositório contém alguns programas desenvolvidos em Java com o objetivo de praticar conceitos básicos de programação, como entrada de dados, estruturas condicionais e lógica.

## 🚀 Projetos incluídos

### 🪖 Alistamento

Arquivo: `Alistamento.java`


Programa que simula um sistema simples de alistamento militar:

* Solicita o nome do recruta
* Recebe peso e altura
* Calcula o IMC
* Informa se o usuário está:

  * Abaixo do peso ideal
  * Dentro do peso ideal
  * Acima do peso ideal

📌 Conceitos utilizados:

* `Scanner`
* Operações matemáticas
* Estruturas condicionais (`if/else`)

---

### ⚔️ Projeto Renascimento

Arquivo: `ProjetoRenascimento.java`


Simula uma verificação de prontidão baseada em condições:

* Verifica se o usuário possui coragem
* Avalia a saúde física
* Exibe uma mensagem baseada nesses critérios

📌 Conceitos utilizados:

* Variáveis booleanas
* Estruturas condicionais

## 🛠️ Como executar os projetos

1. Instale o Java (JDK)
2. Compile o arquivo:

```bash
javac NomeDoArquivo.java
```

3. Execute:

```bash
java NomeDoArquivo
```
